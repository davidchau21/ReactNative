# ReactNative
## Init
```bash
npm i npx -g
npx create-expo-app {AppName}
cd {AppName}
npx expo install react-native-web@~0.19.6 react-dom@18.2.0 @expo/webpack-config@^19.0.0
npm run web
 ```
